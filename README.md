# AppToggler

A lightweight Keyboard Maestro alternative for macOS that lets you toggle applications with customizable hotkeys.

## Features

- Toggle applications on/off with a single keyboard shortcut
- Lightweight menu bar app
- Customizable hotkey configuration
- Built with Swift and SwiftUI for native macOS experience

## Requirements

- macOS 13.0 or later
- Apple Silicon or Intel Mac

## Installation

1. Download the latest release from the [Releases](https://github.com/foyri/AppToggler/releases) page
2. Extract the zip file
3. Move `AppToggler.app` to your Applications folder
4. Launch the app

## Configuration

Configure your hotkeys through the app's interface. Each hotkey can be mapped to toggle a specific application.

## Building from Source

### Requirements

- Xcode 15.0+
- Swift 6.0+

### Build

```bash
./build-app.sh
```

The built app will be in the `dist/` folder.
